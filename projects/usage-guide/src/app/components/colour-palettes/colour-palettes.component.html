Colour Utlities

By installing the @al/design package into your project (see Getting Started section on the Home page), you will have access to SCSS files containing colour variables and utlities enabling you to apply colours to elements you are working on to match UX requirements.

Usage Examples

Import the following into the top of of an SCSS file where you want to use Alert Logic theme colours: 

    
  @import "~@al/design/styles/utility";
  @import "~@al/design/styles/palette";

Inside the palette.scss file are a whole bunch of colour maps, such as.

    
  $al-yellow: (
    500: #FFD54F,
     50: #FFFAEC,
    100: #FFF2C9,
    200: #FFE9A6,
    300: #FFE183,
    400: #FFDB6B,
    600: #FCC101,
    700: #FFD800,
    800: #FFED18,
    900: #FFF36C,
    contrast: (
        50: $al-contrastDark,
      100: $al-contrastDark,
      200: $al-contrastDark,
      300: $al-contrastLight,
      400: $al-contrastLight,
      500: $al-contrastLight,
      600: $al-contrastLight,
      700: $al-contrastLight,
      800: $al-contrastDark,
      900: $al-contrastDark
    )
  );

So to pick out the 500 variation of $al-yellow, you can use the al-color function from utility.scss like so:

    
background: al-color($al-gray, 500);
